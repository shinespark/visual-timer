# VisualTimer

macOS 向けのコンパクトなビジュアルタイマーアプリ。メニューバー常駐型で、セグメントバーと数字表示で残り時間を直感的に確認できます。

## 機能

- **セグメントバー表示** — 20 分割のバーで残り時間を視覚的に表示
- **デジタル表示** — 残り時間を数字で常時表示
- **2 つのタイマーモード**
  - **M1（通常計測）** — カウントダウン / カウントアップを選択可能
  - **M2（リピート）** — 作業・休憩を交互に繰り返すポモドーロ式タイマー
- **音声アラーム** — 半分経過時と終了時にアラーム音を再生
- **通知** — タイマー終了時に macOS 通知を送信
- **設定の永続化** — 前回のモード・時間設定を自動で記憶

## 動作環境

- macOS 13 Ventura 以降
- Xcode 15 以降（ビルド時）

## ビルド方法

```bash
git clone https://github.com/shinespark/visual-timer.git
cd visual-timer
open VisualTimer.xcodeproj
```

Xcode でスキームを `VisualTimer` に設定し、`⌘R` でビルド・実行。

## 使い方

1. アプリを起動するとウィンドウが表示される
2. `▶` ボタンでタイマー開始、`⏸` で一時停止、`↺` でリセット
3. ⚙ アイコンから設定画面を開き、モードや時間を変更して「適用」
